= Create an Uberjar

Uberjar is one method of packaging an application for execution with WildFly Swarm.

.Prerequisites

* A _Maven_-based application with a `pom.xml` file.

.Procedure

. Add the `wildfly-swarm-maven-plugin` to your `pom.xml` by adding a `<plugin>` element, with an `<execution>` element specifying the `package` goal.
+
[source,xml]
----
<plugins>
include::pom.xml[tag=plugin,indent=2]
</plugins>
----

. Perform a normal Maven build:
+
[source]
----
$ mvn package
----

. Execute your uberjar:
[source]
----
$ java -jar ./target/myapp-swarm.jar
----
